Para o projeto utilizaremos:

- node.js lts/hydrogen (LTS - Long Term Support - suporte a longo prazo)
- next@13.1.6
- react@18.2.0
- react-dom@18.2.0

## Dicas para instalação

- Utilizar o nvm para versionar o node
- Define a versão do node.js lts/hydrogen como default pelo comando `nvm alias default lts/hydrogen`

## NextjJS

NEXT Framework para react facilita o desenvolvimento web, renderização do lado do servidor (SSR),
geração de sites estáticos(SSG) e aumenta performance da aplicação.

- File-based routing (Roteamento baseado em arquivos)

- SSR - Server Side Remdering: No next páginas podem ser renderizadas no servidor antes de enviar ao
  cliente, bom para o SEO e o tempo de carregamento.

- SSG - Static Site Generation: pré - renderização de páginas em tempo de construção

- Roteamento automático basedo em arquivos dentro da pasta pages.

- API Routes - criação de rotas de API diretamente no projeto.

## Iniciação do projeto node.js e instalação do Nextjs

Para iniciar um projeto node.js, basta executar o comando:

```
npm init
```

Ou, caso ainda queira pular todas as perguntas, basta executar o comando:

```
npm init -y
```
